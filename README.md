
# 🚀 GSRL SAT HSB Guardian  
**Real-Time Spectral Stability Control for Satellite Systems**

**Author:** Eduardo Hernández Morales  
**Affiliation:** Gamma Systems Research Laboratory (GSRL)  
**IEEE Member**

---

## 📌 Overview

This repository provides a **restricted demonstration** of the  
**GSRL SAT HSB Guardian**, a spectral-based real-time control framework for:

- CubeSats (6U LEO)
- Satellite Attitude Control Systems (ADCS)
- Nonlinear and embedded control applications

The full system has been developed as part of an advanced control architecture
targeting real-time performance in safety-critical environments.

---

## 🔬 Scientific Contribution

- Spectral stability-based control framework (HSB)
- Real-time execution: **< 1 ms**
- Significant improvement over classical LMI-based methods (~142 ms)
- Multi-platform implementation:
  - C++
  - Python
  - MATLAB
  - Julia
  - FPGA (VHDL)
  - PLC (Siemens SCL)
